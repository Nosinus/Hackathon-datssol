# Datsteam offline context pack v2

Contents:

- `docs/input/` — copy these markdown files into your repo's `docs/input/`
- `prompts/` — updated Codex launch prompts
- `repo_root/` — updated `AGENTS.md` template for the repository root

Recommended usage:

1. Copy `docs/input/*.md` into the repo.
2. Copy `repo_root/AGENTS_datsteam_template_v2.md` to repo root as `AGENTS.md`.
3. Launch Codex with `prompts/codex_datsteam_prompt_only_v2.txt` or the full master prompt file.
